## Tree
Tree utility in Clojure. Renders file tree in the terminal.

Usage:
```tree <directory> [file filter regex]```

Sample rendering:
```
.
├─ .gitignore
├─ .lein-repl-history
├─ README.md
├─ project.clj
├─ src
│  └─ tree
│     └─ main.clj
└─ target
   ├─ classes
   └─ stale
      └─ dependencies
```
